Striper Dice Assets Repository

This repository contains the assets for the Striper Dice Roller application.

Available Assets:

  • dice-box.es.min.js - Main dice box module
  • world.offscreen.min.js - Worker script for physics calculations
  • assets/ - Directory containing themes and 3D models

These assets are used by the Striper Dice Roller application. For more information, visit the main application.
